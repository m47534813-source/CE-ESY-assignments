# CE-ESY-assignments
Circular Buffer Implementation

 فكرة البرنامج

هذا البرنامج يطبق مفهوم **Circular Buffer** باستخدام لغة C.
وهو مخزن (مصفوفة) نكتب ونقرأ منه بشكل دائري.



طريقة العمل

عندنا مؤشرين:

  * `h` للقراءة
  * `t` للكتابة
  ومتغير `c` لعدد العناصر



العمليات

الإدخال (push)

* نضيف العنصر في مكان `t`
* نحرك `t` خطوة للأمام بشكل دائري
* إذا المخزن ممتلئ تظهر:

  
  overflow
  

 الإخراج (pop)

* نقرأ من مكان `h`
* نحرك `h` خطوة للأمام
* إذا المخزن فارغ تظهر:

  
  underflow
  



داخل main

* نقرأ اسم من المستخدم
* نضيف `"CE-ESY"` للاسم
* نخزن الأحرف في الـ buffer
* نرجع نطبعهم مرة ثانية



ملاحظة

اخترت حجم صغير للمخزن حتى تظهر حالة overflow أثناء التشغيل.
